# multiple-management

## 简介

1. VueCli构建的多页面vue应用
2. 各个应用的router及页面相互隔离
3. 分为Login应用、AppList应用及各个业务应用
4. 应用内部使用路由需要使用Hash模式而不是History模式

## Project setup

```
npm install
```
